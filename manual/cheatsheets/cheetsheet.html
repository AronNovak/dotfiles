If you find your data file is getting too long to be manageable, go ahead and split it into multiple YAML files. cheater can take a directory as its -i/--in argument
cheater -i foo/

Fill out the contents of your cheat sheet in this YAML file.

cheater -i foo.yml |

Create a blank cheat sheet skeleton

cheater init -o foo.yml |
